$freshChatProvider
service in module freshdesk.chat

Description

The provider for the $freshChat service. Allows boot-time configuration of the FreshChat widget.

Methods

  • setAccount(account)

    Set the FreshDesk account name. Internally, this will set the link to your support portal.

    Parameters
    ParamTypeDetails
    accountstring

    Your account name

  • setAccountUrl(url)

    Set the URL to your FreshDesk support portal.

    Parameters
    ParamTypeDetails
    urlstring

    Your account URL

  • setCredentials(id, [secret])

    Set the FreshChat credentials.

    Parameters
    ParamTypeDetails
    idstring

    The FreshChat client id

    [secret]string

    The FreshChat client secret
